OrthoNova Clinic - Offline Version

This is the offline version of the OrthoNova Poly Clinic management system. It uses your browser's localStorage instead of a remote database, allowing it to run completely offline.

How to Use the Offline Version

Step 1: Navigate to the Offline Folder

Open your file explorer and navigate to the orthnova-offline folder.

Step 2: Build the Application (if not already built)

If you haven't built the application yet, open a terminal/command prompt in the orthnova-offline folder and run:

npm run build

Step 3: Open the Application

Navigate to the build subfolder and double-click on index.html to open it in your default browser.

Alternatively, you can right-click on index.html and choose "Open with" to select your preferred browser.

Login Credentials

On first run, the system creates a sample admin user:

  • Username: admin
  • Password: admin123

You can change this password or create new users within the application.

Important Notes

  • All data is stored in your browser's localStorage and will persist between sessions
  • Data is only available in the browser where it was created
  • To clear all data, use your browser's developer tools to clear localStorage
  • This offline version has all the same features as the online version

Troubleshooting

If the Application Doesn't Load

Some browsers may block local files from accessing localStorage. If this happens:

  1. Try using a different browser (Chrome, Firefox, or Edge recommended)
  2. Use a simple HTTP server instead of opening the file directly

Using a Local Server (Alternative Method)

If opening the file directly doesn't work, you can use a simple HTTP server:

  1. Install Node.js if you haven't already
  2. Open a terminal in the orthnova-offline/build folder
  3. Run: npx serve -s
  4. Open your browser and go to http://localhost:3000
