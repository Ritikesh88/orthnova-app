This is a design-only, non-destructive UI mockup for the OrthoNova clinic management system receptionist workflow. The interface demonstrates the following features:

  • Receptionist Dashboard with Today's Overview
  • Patient Registration Screen

  • Vitals Section with Placeholders
  • Diagnosis, Medicines, and Advice Sections
  • Clinic Billing Interface
  • Print Preview Functionality

Note: This is a visual demonstration only. All interactions are simulated for UX validation purposes.
